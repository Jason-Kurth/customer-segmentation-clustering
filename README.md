# Customer Segmentation with K-Means

## Problem
Businesses often have large customer datasets but lack clear segmentation to guide targeted marketing and strategy.

This project applies unsupervised machine learning to identify meaningful customer groups.

## Method
- Data cleaning and feature engineering
- Dimensionality reduction using PCA and t-SNE
- Optimal cluster identification using elbow and silhouette methods
- K-Means clustering to generate segments

## Tools
Python, Pandas, scikit-learn, PCA, t-SNE, K-Means

## Outputs
- Jupyter Notebook with full analysis
- Project summary report

- `customer_segmentation_analysis.ipynb` — full analysis
- `project_summary_report.pdf` — written summary
